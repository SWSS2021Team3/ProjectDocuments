## ACS : Attendance Check System (PC)

## FRS : Face Recognition System (Jetson Nano)
### Install
- cat FRS.tar* | tar xvf -
- unzip FRS.zip
- cd deploy
### Run Camera mode (has some problem with closing camera)
- ./run.sh
### Run Play Record mode (can be switched to Camera mode)
- ./run-mov.sh
